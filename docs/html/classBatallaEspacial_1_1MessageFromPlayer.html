Constructor. Toma toda la información de un jugador y construye el mensaje, guardándolo en msg

Parámetros:
_id: Id
_name: Nombre
_escudo: Valor de escudo actual
_x: Posición X actual
_y: Posición Y actual
_dir: Dirección actual
_vm: Valor del byte VM. 'V' si está vivo, M si está muerto
_power: Valor de potencia. Si no está disparando, es DisparoEnemigo.VALORESPECIAL.
_lx: Posición X actual del laser. Si no está disparando, es DisparoEnemigo.VALORESPECIAL.
_ly: Posicion Y actual del laser. Si no está disparando, es DisparoEnemigo.VALORESPECIAL.
_moneda: Id de la moneda que agarró antes de enviar el mensaje. -1 si no agarró ninguna.
_idAsesino: Número entre 0 y 3 del id enemigo que lo destruyó, o -1 si nadie lo destruyó.
_fm: Framerate del jugador.
