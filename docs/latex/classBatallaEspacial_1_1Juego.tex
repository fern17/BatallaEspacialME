\subsection{Descripción detallada}
Clase \hyperlink{classBatallaEspacial_1_1Juego}{Juego}. Se encarga de toda la lógica de la aplicación una vez que se está jugando. Captura el teclado, calcula colisiones, actualiza posiciones, arma mensajes, envia y recibe mensajes y sincroniza los jugadores.
